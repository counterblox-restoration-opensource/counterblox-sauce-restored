Here is the open source release of Counter Blox: Sauce - Restored, use it according to the license.
You will need Roblox Studio to open the game file.

The game Counter Blox: Sauce - Restored is the restored version of Counter Blox: Sauce (also known
as CB:S or CB: Sauce), which was a FPS game made in 2015. Because of the FilteringEnabled
feature that Roblox added, this game has been broken and not fixed at all.

Please also note that this restored version comes with a lot of added features, such as trading,
skins, community market translations and much more. There is already a setting that everyone can
choose to disable all the new added feature.
